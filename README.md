

<h1>🎨 Color Game</h1>

<div>
  <span class="badge badge-primary">React</span>
  <span class="badge badge-secondary">TailwindCSS</span>

</div>

<p>A fast-paced, engaging color matching game built with React and TailwindCSS. Test your color recognition skills and compete against time!</p>

<h2>🎯 Features</h2>

<ul class="feature-list">
  <li>Real-time color matching challenges</li>
  <li>60-second time limit per game</li>
  <li>Score tracking system (+10 for correct, -5 for wrong)</li>
  <li>Responsive design for all devices</li>
  <li>30+ contrasting colors</li>
  <li>Visual feedback for correct/incorrect answers</li>
  <li>Clean, modern UI with animations</li>
</ul>

<h2>🚀 Getting Started</h2>

<p>Follow these steps to run the game locally:</p>

<pre><code># Clone the repository
git clone https://github.com/gabrielisaacs/color-game.git

# Navigate to project directory
cd color-game

# Install dependencies
npm install

# Start the development server
npm run dev</code></pre>

<h2>🎮 How to Play</h2>

<ul class="feature-list">
  <li>Click "NEW GAME" to start</li>
  <li>Match the color shown in the top button</li>
  <li>Score +10 points for correct matches</li>
  <li>Lose 5 points for wrong matches</li>
  <li>Complete as many matches as possible in 60 seconds</li>
</ul>
